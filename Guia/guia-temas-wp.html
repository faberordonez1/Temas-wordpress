Creando template con Bootstrap

  1. Para crear el menu vamos a la documentación de Bootstrap en la seccion componentes>nav y copiamos la plantilla base
  2. Abrimos el archivo HTML y creamos un div contenedor con la clase (container-fluid), dentro de este pegamos la plantilla base y cambiamos el color del menu usando la clase (navbar-dark bg-dark tambien aplicamos la clase (bg-dark) al div contenedor
  3. eliminamos el buscar y a los li les quitamos la clase del estilo (disabled) y copiamos los li hasta tener 4 items simples,
  4. Para que los items li se alinien a la derecha cambiamos la clase del ul asi (ms-auto por me-auto)
  5. Agregamos la imagen en dentro de la etiqueta de enlace (a)

Pagina estatica HTML Inicial

Iniciamos con 3 paginas estaticas creadas con HTML y Bootstrap, estas estaran enlazadas entre si, estas luego se modificaran para crear el tema

  • index.html: Página principal
  • single.html: Página de entradas
  • page.html: Página de destino de los items del menú

Convertimos el archivo index.html en index.php el cual a su vez se divide en 3 archivos header, entradas y footer

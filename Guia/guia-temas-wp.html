Creando pagina estatica con Bootstrap

Nav

  1. Para crear el menu vamos a la documentación de Bootstrap en la seccion componentes>nav y copiamos la plantilla base
  2. Abrimos el archivo HTML y creamos un div contenedor con la clase (container-fluid), dentro de este pegamos la plantilla base y cambiamos el color del menu usando la clase (navbar-dark bg-dark tambien aplicamos la clase (bg-dark) al div contenedor
  3. eliminamos el buscar y a los li les quitamos la clase del estilo (disabled) y copiamos los li hasta tener 4 items simples,
  4. Para que los items li se alinien a la derecha cambiamos la clase del ul asi (ms-auto por me-auto)
  5. Agregamos la imagen en dentro de la etiqueta de enlace (a)

A continuacion la imagen del codigo citando algunos de los items anteriores

Formulario

Titulo

  1. Crear div contenedor con la clase container fluid
  2. Enlazar el archivo style.css
  3. Creamos otro div interno con la clase text-center
  4. Dentro de este div crear un h1 con la clase display-4 y un texto aleatorio
  5. Para centrar el texto usamos flexbox aplicando al contenedor principal las clases d-flex justify-content-center flex-column align-items-center (utilidades botstrapt)
  6. Para que el h1 tome el color blanco aplicamos la clase text-white al h1

Input

  1. Para hacer el Formulario abrimos la etiqueta form
  2. Dentro del form creamos un div con la clase form-group para el primer input
  3. Dentro del div creamos un input para el nombre de tipo texto y le asignamos la clase form-control
  4. Para el campo email realizamos el paso anterior, pero para el campo e-mail
  5. Creamos un tercer input de tipo submit para el boton y le asignamos la clase btn btn-primary para darle estilo ademas adicionamos el atributo value enviar
  6. Para poner el formulario en linea agregamos flexbox a la etiqueta form con la clase d-flex

Pagina estatica HTML Inicial

Iniciamos con 3 paginas estaticas creadas con HTML y Bootstrap, estas estaran enlazadas entre si, estas luego se modificaran para crear el tema

  • index.html: Página principal
  • single.html: Página de entradas
  • page.html: Página de destino de los items del menú

Convertimos el archivo index.html en index.php el cual a su vez se divide en 3 archivos header, entradas y footer

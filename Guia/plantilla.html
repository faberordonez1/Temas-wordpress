Introducción

Iniciamos con 3 paginas estaticas creadas con HTML y Bootstrap, estas estaran enlazadas entre si, estas luego se modificaran para crear el tema

  • index.html: Página principal
  • single.html: Página de entradas
  • page.html: Página de destino de los items del menú

Convertimos el archivo index.html en index.php el cual a su vez se divide en 3 archivos header, entradas y footer

Páginas estaticas con Bootstrap

Index.html

Nav

  1. Para crear el menu vamos a la documentación de Bootstrap en la seccion componentes>nav y copiamos la plantilla base
  2. Abrimos el archivo HTML y creamos un div contenedor con la clase (container-fluid), dentro de este pegamos la plantilla base y cambiamos el color del menu usando la clase (navbar-dark bg-dark tambien aplicamos la clase (bg-dark) al div contenedor
  3. eliminamos el buscar y a los li les quitamos la clase del estilo (disabled) y copiamos los li hasta tener 4 items simples,
  4. Para que los items li se alinien a la derecha cambiamos la clase del ul asi (ms-auto por me-auto)
  5. Agregamos la imagen en dentro de la etiqueta de enlace (a)

Formulario

Titulo
  1. Crear div contenedor con la clase container fluid
  2. Enlazar el archivo style.css y aplicamos la clase suscribir del style.css al contenedor creado
  3. Creamos otro div interno con la clase text-center
  4. Dentro de este div crear un h1 con la clase display-4 y un texto aleatorio
  5. Para centrar el texto usamos flexbox aplicando al contenedor principal las clases d-flex justify-content-center flex-column align-items-center (utilidades botstrapt)
  6. Para que el h1 tome el color blanco aplicamos la clase text-white al h1
Input
  1. Para hacer el Formulario abrimos la etiqueta form
  2. Dentro del form creamos un div con la clase form-group para el primer input
  3. Dentro del div creamos un input para el nombre de tipo texto y le asignamos la clase form-control
  4. Para el campo email realizamos el paso anterior, pero para el campo e-mail
  5. Creamos un tercer input de tipo submit para el boton y le asignamos la clase btn btn-primary para darle estilo ademas adicionamos el atributo value enviar
  6. Para poner el formulario en linea agregamos flexbox a la etiqueta form aplicando la clase d-flex
  7. Para centrar los campos del formulario en responsive aplicamos las clases de flexbox flex-column flex-sm-row a la etiqueta form
  8. Para centrar los elementos del nav en responsive aplicamos la clase text-center al ul del nav
  9. Para separar los input en desktop agregamos un espaciado a la derecha añadiendo la clase me-sm-3 al div con la clase form-group
  10. Para separar los input verticalmente en responsive agregamos un espaciado añadiendo la clase mb-3 al div con la clase form-group

Contenido

  5. Agregamos la ruta de la imagen de la card https://loremflickr.com/940/788/
  6. A la etiqueta img aplicamos la clase img-fluid para que se adapte en responsive
  7. Agregamos un espaciado inferior junto a la clase card aplicando mb-3
  8. Agregamos un texto utilizando el div con la clase card-footer text-muted de los templates de bootstrap se pega junto a card-body y cambiamos el texto por fecha/etiquetas/categorias/Autor
  9. Copiamos las card completa 2 veces mas

Página de aterrizaje de las pestañas de menu

  1. Creamos un nuevo archivo con el nombre page.html
  2. Copiamos el codigo ya creado en el index.html desde el inicio hasta fin suscribir y lo pegamos en page.html
  3. De nuevo copiamosy pegamos pero esta ves desde el comentario de inico del footer hasta fin del cierre de la etiqueta principal html
  4. Despues del cierre de fin suscribir hacemos un nuevo comentario contenido menu y fin contenido
  5. Dentro de los comentarios creamos un nuevo div con la clase container my-5 y el espaciado en y
  6. Dentro del div creamos un H2 con texto para el titulo de la pestaña
  7. Creamos un nuevo div con la clase text-justify para justificar el texto contenido
  8. Dentro del nuevo div agregamos los parrafo con texto contenido
